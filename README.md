Project Overview

Modern sports organizations increasingly rely on analytics to improve decision-making; however, the success of these analytics depends not only on technical accuracy but also on how effectively insights are communicated and implemented. As emphasized in Chapter 7, many strong analytics ideas fail due to poor integration into organizational processes.

This project introduces the Player Impact Efficiency Score (PIES), a simplified yet comprehensive metric designed to capture a player’s overall contribution to team performance. The objective is to transform complex performance data into an accessible and actionable format for decision-makers.

Decision-Making Problem

Sports organizations often face challenges in evaluating player performance using fragmented statistics. Traditional metrics such as goals or assists fail to fully capture a player’s overall impact, leading to incomplete or biased decisions.

This project addresses the problem of disconnected performance indicators by proposing a unified metric that supports more consistent and informed decision-making. By reducing analytical complexity, the tool enhances clarity for coaches, managers, and executives (Alamar, 2024).

Proposed Analytics Approach

The Player Impact Efficiency Score integrates multiple dimensions of player performance, including offensive output, defensive contributions, and contextual match factors. Data sources may include event-based match statistics, player tracking data, and situational variables such as match intensity.

Rather than focusing on computation, this project emphasizes how analytics ideas can be structured and communicated effectively. The approach aligns with the concept that analytics must be actionable and relevant to decision-makers to create value (Alamar, 2024).

Use by Decision Makers

The tool is designed for practical use by coaches, team managers, and front office executives. Coaches can apply the metric during matches to guide substitutions and tactical adjustments. Managers can evaluate player consistency over time, while executives can incorporate the metric into recruitment and performance evaluation decisions.

By simplifying complex data into a single score, the tool reduces cognitive burden and improves the speed and quality of decision-making.

Connection to Chapter 7

This project currently represents the creative phase of the analytics innovation framework described in Chapter 7. At this stage, the focus is on defining a meaningful and relevant analytics idea that addresses a real organizational problem.

The emphasis is on clarity, relevance, and potential value rather than technical implementation, which will be explored in later phases such as prototyping and engagement.

Prototype Enhancement

In the prototype phase, the original model is enhanced by incorporating real-time performance updates. This modification allows the Player Impact Efficiency Score to adjust dynamically during matches rather than relying solely on post-game analysis.

The enhancement improves decision-making by enabling immediate responses to changes in player performance. For example, a decline in defensive contribution can be identified instantly, allowing coaches to make timely substitutions or tactical adjustments.

Prototype Evaluation

The prototype introduces meaningful improvements by increasing the timeliness and relevance of the analytics tool. However, its integration depends on feedback from decision-makers.

If coaches and managers find the real-time updates intuitive and actionable, the enhancement should be adopted. Conversely, if the added complexity reduces usability, further refinement would be necessary before full implementation.

Integration Decision

The prototype enhancement will be merged into the main branch because it significantly improves the practical value of the analytics tool. Real-time insights align closely with the needs of sports organizations, where rapid decision-making is essential.

This decision reflects the transition from prototyping to the early stages of implementation within the innovation framework.

Reflection on Innovation and Version Control

Version control systems such as GitHub play a critical role in supporting analytics innovation. The use of branches allows teams to experiment with new ideas in a controlled environment without disrupting the main project. This reduces risk and encourages creativity.

Additionally, GitHub enhances communication and transparency by documenting changes and enabling collaboration. Decision-makers can review the evolution of analytics ideas, provide feedback, and build confidence in their adoption.

This workflow aligns with Chapter 7 by supporting the progression from creative idea generation to prototyping, engagement, and eventual implementation.
